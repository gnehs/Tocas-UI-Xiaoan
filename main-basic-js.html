介紹一個基本的導覽

在 Tocas Xiaoan 中，一些功能同時有輸出在網頁中及輸出在 console 裡的版本，若您要輸出在 console 中，在後方加上 _con 即可

我們已經在文件中加上標籤，方便尼知道該功能可以用在哪裡了❤️

原始碼
/*底下程式碼會輸出在網頁上*/
xiaoan.earlySpring404()
/*底下程式碼會輸出在 console 裡*/
xiaoan.earlySpring404_con()

每日洨安
網頁中
Console
輸出洨安在網路上公開講過的話，尼甚至可以指定數量及限制字數。

原始碼
/* 預設輸出一個，且無指定字數 */
xiaoan.daily()
/* 指定輸出數量為 3 個，且限制字數小於 15 */
/* xiaoan.daily("輸出數量","限制字數") */
xiaoan.daily("3","15")


基本洨安
網頁中
輸出洨洨安，您可以指定樣式，超棒 Der。

原始碼
xiaoan.basic()
/* 樣式請參考 CSS 裡的基本章節 */
/* xiaoan.basic("樣式") */
xiaoan.basic("horny love big")


點心條
網頁中
無需使用 Tocas JS 的替代方案，而且不用新增點心條的元素

原始碼
/* 底下的點心條將持續一秒(預設) */
snackbar("你點了按鈕，真是太棒了，有機會抽獎獲得 iPhone8 喔")
/* 底下的點心條將持續五秒，修改數字來更改持續時間 */
snackbar("我會持續五秒喔",5)
