引用
下載並解壓縮後，保持資料夾結構，丟到你的網頁目錄，即可開始使用！

原始碼
<!-- 洨洨安 CSS -->
<link rel="stylesheet" href="ts.xiaoan.min.css">
<!-- 洨洨安 JS -->
<script src="ts.xiaoan.min.js"></script>




外部引用
不放在自家伺服器的方法，這裡有兩個位置可以自由選用

原始碼
<!-- RawGit  -->
<!-- 洨洨安 CSS -->
<link rel="stylesheet" href="//cdn.rawgit.com/gnehs/Tocas-UI-Xiaoan/master/ts.xiaoan.min.css">
<!-- 洨洨安 JS -->
<script src="//cdn.rawgit.com/gnehs/Tocas-UI-Xiaoan/master/ts.xiaoan.min.js"></script>


原始碼
<!-- GitHub Page -->
<!-- 洨洨安 CSS -->
<link rel="stylesheet" href="//xiaoan.tocas-ui.com/ts.xiaoan.min.css">
<!-- 洨洨安 JS -->
<script src="//xiaoan.tocas-ui.com/ts.xiaoan.min.js"></script>
